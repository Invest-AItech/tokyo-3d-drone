# TOKYO 3D Drone

Multi-waypoint cinematic drone camera over Tokyo (PLATEAU 3D).

3 部作の第 3 部（総称: TOKYO 3D Lab）。

- 第 1 部: TOKYO 3D View — 1 点指定で周辺建物表示
- 第 2 部: TOKYO 3D Route — 2 点ルートをドローン視点で自動飛行
- **第 3 部 (本リポ): TOKYO 3D Drone** — 多点 + 高さ + ユーザー設計のカメラワーク
